Using the Newton-Raphson Root Finder

To use the Newton-Raphson root finder, enter a valid mathematical function of \(x\) in the input field. You can use standard mathematical notation, such as for the function \(f(x)\) =.

Provide an initial guess \(x_0\) for the root and specify a tolerance level \(Îµ\) to determine the accuracy of the result. Click the "Find root" button to start the computation.

Mathematical Function Format

The function supports the following expression formats:

  • Arithmetic operations
    • Addition, subtraction, multiplication, and division
    • Exponentiation
      • Can use ** or ^
      • Limited to one digit
  • Algebraic operations (with the variable \(x\))
  • Trigonometric functions
    • \(sin, cos, tan, cot, sec, cosec\)
  • Logarithms
    • \(log, ln\)

Understanding the Results

The results section will display the approximate root found by the Newton-Raphson method, along with the number of iterations taken to converge to that root. If the method fails to converge, an appropriate message will be shown.
